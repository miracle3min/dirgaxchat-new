# DirgaX Chat

**Smart Chat In Your Hand!**

Asisten AI pribadi yang cerdas, cepat, dan aman. Deploy dalam satu klik.

---

## ✨ Fitur Utama

- 🧠 Multi-model AI support
- 🖼️ Multimodal — gambar, video, audio, dokumen
- 🎙️ Talk Mode — ngobrol pakai suara
- 🔌 Plugin System — Web Search, Weather, dll
- 🎨 Claymorphism UI — modern & responsive
- 📱 PWA + Desktop support
- 🌐 Multi-bahasa (default: Bahasa Indonesia)
- 🔒 Privasi — semua data tersimpan lokal
